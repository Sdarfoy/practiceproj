```markdown
# Создание документов

Программа для работы с документами

## О программе

Программа умеет создавать три вида документов, отображает их список, позволяет сохранить документ в текстовый файл, загрузить документ из текстового файла

### Установка

1. Клонируйте репозиторий с помощью команды `git clone https://github.com/Sdarfoy/practiceproj.git`
2. Перейдите в директорию проекта `cd practiceproj`
3. Установите необходимые зависимости `npm install`

```
